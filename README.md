## 📋 What This Tool Does

- Fetches real-time NIFTY option chain data
- Calculates Open Interest (OI) percentages
- Tracks market sentiment (BULLISH, BEARISH, RISKY, etc.)
- Displays support/resistance levels
- Shows buy/sell targets based on predefined logic
